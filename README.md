# Simple Code Editor

A lightweight, modern code editor built with Next.js and React. Features a file explorer, split-pane layout, and dark mode support.

This is a frontend-only project. Data persists in browser memory only (no backend/database).

## Setup

```sh
npx create-next-app .
npx shadcn@latest init
npx shadcn@latest add --all

# use https://github.com/AntonioErdeljac/polaris-assets/blob/main/globals.css

# Add dark mode: https://ui.shadcn.com/docs/dark-mode/next
npm install next-themes

# Add split-pane https://www.npmjs.com/package/allotment
npm i allotment

# File & Folder Icons: https://www.npmjs.com/package/@react-symbols/icons
npm i @react-symbols/icons
```
